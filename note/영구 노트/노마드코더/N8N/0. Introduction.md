## 0.3 What is N8N?
N8N은 Low-code 기반의 **워크플로우 자동화 플랫폼**입니다. 시각적인 인터페이스를 통해 다양한 애플리케이션과 시스템을 연결하여 자동화된 워크플로우를 구축할 수 있습니다.

### 주요 특징
- Low-code & 유연성: 드래그 앤 드롭 방식의 쉬운 사용성과 함께, 필요할 경우 JavaScript나 Python 코드를 사용하여 복잡하고 맞춤화된 로직을 추가할 수 있습니다.
- 오픈소스 (Fair-Code): Fair-code 라이선스로 배포되며, 사용자가 직접 서버에 호스팅(Self-hosted)하여 데이터와 배포 환경에 대한 완전한 통제권을 가질 수 있습니다.
- 노드(Node) 기반: 각 노드는 특정 앱과의 연결 또는 액션을 수행하며, 이 노드들을 연결하여 자동화 파이프라인(워크플로우)을 만듭니다.
- 다양한 통합: 400개 이상의 사전 구성된 통합(Integrations)을 지원하며, HTTP Request 노드를 통해 모든 REST API에 연결할 수 있습니다.
