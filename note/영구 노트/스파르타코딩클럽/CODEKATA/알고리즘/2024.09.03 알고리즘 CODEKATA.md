## 짝수의 합

### 문제 설명

정수 `n`이 주어질 때, `n`이하의 짝수를 모두 더한 값을 return 하도록 solution 함수를 작성해주세요.

##### 제한사항

- 0 < `n` ≤ 1000

#### 입출력 예

| n   | result |
| --- | ------ |
| 10  | 30     |
| 4   | 6      |

---

## 내 답변

```python
def solution(n):
    sum = 0
    for num in range(1, n+1):
        sum = sum + num if num % 2 == 0 else sum + 0

    return sum
```
