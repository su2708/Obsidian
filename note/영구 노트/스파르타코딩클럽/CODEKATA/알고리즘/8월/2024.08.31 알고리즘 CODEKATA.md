## 두 수의 합

### 문제 설명

정수 `num1`과 `num2`가 주어질 때, `num1`과 `num2`의 합을 return하도록 soltuion 함수를 완성해주세요.

##### 제한사항

- -50,000 ≤ `num1` ≤ 50,000
- -50,000 ≤ `num2` ≤ 50,000

#### 입출력 예

|num1|num2|result|
|---|---|---|
|2|3|5|
|100|2|102|

---

## 내 답변

```python
def solution(num1, num2):
    answer = num1+num2
    return answer
```
