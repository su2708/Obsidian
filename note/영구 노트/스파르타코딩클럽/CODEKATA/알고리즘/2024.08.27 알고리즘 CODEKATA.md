## 두 수

정수 `num1`, `num2`가 매개변수 주어집니다. `num1`과 `num2`를 곱한 값을 return 하도록 solution 함수를 완성해주세요.
##### 제한사항
- 0 ≤ `num1` ≤ 100
- 0 ≤ `num2` ≤ 100

#### 입출력 예

| num1 | num2 | result |
| ---- | ---- | ------ |
| 3    | 4    | 12     |
| 27   | 19   | 513    |

---

## 내 답변

```python
def solution(num1, num2):
    return num1*num2
```
